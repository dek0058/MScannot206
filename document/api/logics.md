# 📚 Logic API Reference

Logic API는 게임의 다양한 기능을 제어하고 데이터를 관리하는 핵심 모듈들의 집합입니다.

각 로직은 특정 기능을 담당하며, 게임 내에서 발생하는 다양한 이벤트와 상호작용을 처리합니다.

아래는 주요 Logic API들의 목록과 간단한 설명입니다.

| Logic Name | Description |
| :--- | :--- |
| [**BackendLogic**](./logic/BackendLogic.md) | 백엔드 서버의 통신에 필요한 메소드 및 프로퍼티를 제공합니다. |
| [**ChannelLogic**](./logic/ChannelLogic.md) | 채널 관리 및 플레이어 채널 이동에 필요한 메소드를 제공합니다. |
| [**CharacterAvatarLogic**](./logic/CharacterAvatarLogic.md) | 캐릭터 아바타 관련 기능을 제공합니다. |
| [**CharacterDef**](./logic/CharacterDefLogic.md) | 캐릭터 정의 및 관련 상수를 관리하는 프로퍼티를 제공합니다. |
| [**CharacterLogic**](./logic/CharacterLogic.md) | 캐릭터 관리 관련 메소드 및 프로퍼티를 제공합니다. |
| [**DefLogic**](./logic/DefLogic.md) | 다양한 정의 및 상수를 관리하는 프로퍼티를 제공합니다. |
| [**ErrorLogic**](./logic/ErrorLogic.md) | 오류 처리에 대한 메소드를 제공합니다. |
| [**GameClientInputLogic**](./logic/GameClientInputLogic.md) | 클라이언트 입력 처리 관련 기능을 제공합니다. (작성 중) |
| [**GameClientLogic**](./logic/GameClientLogic.md) | 유저의 클라이언트를 관리하는 메소드를 제공합니다. |
| [**GameClientUILogic**](./logic/GameClientUILogic.md) | UI 처리에 대한 메소드 및 프로퍼티를 제공합니다. |
| [**GameDBLogic**](./logic/GameDBLogic.md) | MSW 데이터베이스와의 통신에 필요한 메소드를 제공합니다. |
| [**GameplayLogic**](./logic/GameplayLogic.md) | 게임 플레이 진행 및 상태 관련 기능을 제공합니다. (작성 중) |
| [**GameStaticsLogic**](./logic/GameStaticsLogic.md) | 게임 내 편리한 함수를 제공합니다. |
| [**GameServerLogic**](./logic/GameServerLogic.md) | 유저의 서버 요청에 대한 처리에 필요한 메소드를 제공합니다. |
| [**GameTableLogic**](./logic/GameTableLogic.md) | 게임 테이블 데이터를 관리하는 메소드를 제공합니다. |
| [**InventoryLogic**](./logic/InventoryLogic.md) | 인벤토리 관리에 필요한 메소드 및 프로퍼티를 제공합니다. |
| [**ItemLogic**](./logic/ItemLogic.md) | 아이템 관리에 필요한 메소드 및 프로퍼티를 제공합니다. |
| [**LoginLogic**](./logic/LoginLogic.md) | 로그인 프로세스에 필요한 메소드를 제공합니다. |
| [**StringLogic**](./logic/StringLogic.md) | 문자열 처리에 대한 유틸리티 메소드를 제공합니다. |
| [**TransitionLogic**](./logic/TransitionLogic.md) | 씬 전환에 필요한 메소드를 제공합니다. |
| [**TweenLogic**](./logic/TweenLogic.md) | 트윈 관련 메소드 및 프로퍼티를 제공합니다. |
| [**UserLogic**](./logic/UserLogic.md) | 유저 정보 관리에 필요한 메소드 및 프로퍼티를 제공합니다. |
| [**UserProxy**](./logic/UserProxyLogic.md) | 유저 컴포넌트 및 인게임 내 유저 데이터 송수신을 관리하는 메소드를 제공합니다. |
